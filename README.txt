========================================
  盘策 · 炒股助手 — 使用说明
========================================

【一、电脑上打开（最简单）】
  直接双击 "index.html" 文件，浏览器就会打开 App。
  按 F12 → 点左上角的手机图标，可以模拟手机屏幕看效果。

【二、安装到手机上（iOS / 安卓都支持）】

  第 1 步：部署到免费网址
    方法 A：GitHub Pages（推荐，完全免费）
      1. 注册 GitHub 账号：https://github.com
      2. 新建仓库（New Repository），名字随便取，比如 pance
      3. 把 "盘策" 文件夹里的所有文件上传到仓库
      4. 进入仓库 → Settings → Pages → Branch 选 main → Save
      5. 等 1 分钟后，会显示网址：https://你的用户名.github.io/pance

    方法 B：Vercel（也免费）
      1. 注册 https://vercel.com （用 GitHub 登录）
      2. 点 New Project → 导入你的仓库 → Deploy
      3. 自动生成网址：https://xxx.vercel.app

    方法 C：Netlify（也免费）
      1. 注册 https://netlify.com
      2. 把文件夹拖到部署页面即可
      3. 自动生成网址

  第 2 步：手机浏览器打开网址
    - 复制上面的网址，发到微信里
    - 在微信里点链接 → 选择「在浏览器中打开」

  第 3 步：安装到桌面
    【安卓手机】
      Chrome 浏览器打开网址 → 底部会自动弹出「添加到主屏幕」→ 点确定
      如果没有弹出，点右上角 ⋮ → 添加到主屏幕
    【苹果手机】
      Safari 浏览器打开网址 → 点底部中间的「分享」按钮（方框+箭头）
      → 往下滑 → 点「添加到主屏幕」→ 点右上角「添加」

  安装完成后，桌面会出现「盘策」图标，点开就是全屏 App！

【三、功能说明】

  1. 大盘情绪（首页）
     - 自动显示上证指数和冰山指数评分
     - 仪表盘直观展示市场情绪：绿色悲观 → 灰色中性 → 红色乐观
     - 分数规则：3000 点 = 60 分，每涨 100 点 +1 分，每跌 100 点 -1 分
     - 每 5 秒自动刷新

  2. 异动预警
     - 搜索股票代码（如 600519 贵州茅台）→ 点添加
     - 勾选预警类型：涨停 / 跌停 / 急拉 2% / 急跌 2%
     - 满足条件时页面弹窗提醒
     - 微信推送设置（可选）：
       打开 https://sct.ftqq.com → 微信扫码登录
       → 获取 SendKey → 粘贴到 App 的设置框 → 点保存
       之后预警就会推送到你微信！

  3. 持仓评级
     - 添加你的持仓股票代码
     - 自动显示周涨跌幅、月涨跌幅
     - 自动评级：A 级（强势）→ E 级（严重破位）
     - 顶部显示组合整体评级

  4. 仓位安全
     - 输入总资产和已用资金
     - 自动计算仓位比例
     - 四级安全标准：
       ≤70% = 安全稳健（绿色）
       71-80% = 适中控制（蓝色）
       81-90% = 风险偏高（橙色）
       91-100% = 高度谨慎（红色）

【四、数据来源】
  股票数据来自东方财富（East Money）免费公开 API，约 3-5 秒延迟。
  数据仅供个人参考，不构成投资建议。

【五、常见问题】

  Q: 搜索股票没反应？
  A: 检查股票代码是否正确。上海股票 6 开头（如 600519），深圳股票 0/2/3 开头。

  Q: 预警不推微信？
  A: 检查是否已填入 Server酱 SendKey，且 SendKey 没有过期。

  Q: 大盘数据不更新？
  A: 非交易时间（周末/节假日/收盘后）数据不会变动，是正常的。

  Q: 周涨跌幅/月涨跌幅显示不准？
  A: 数据需要等 K 线 API 返回，首次添加可能需要等几十秒。
